

Los componentes pueden ser sin Estado "Stateless" o con Estado "Statefull"


Los componentes con Estado o Statefull son aquellos que te permiten modificar su informacion interna a partir de

eventos o interacciones con el mismo

Los componentes Stateless le brindan la informacion al momento de ser creados y no le permiten modificarlo luego


Los componentes Statefull permiten reaccionar a eventos , estos dejan de ser funciones nativas de JS, ahora pasan

a ser clases.




Para crear un componente con estado y volver las funciones a clases:

        import React, {Component} from 'react'

        class Saludar extends Component{

            render(){
                
                return(
                    <h1>Hello </h1>
                )
            }
        }



Son los componentes que estan atentos a los eventos que los rodean y pueden reaccionar a los mismos 

En funcion de esos eventos el componente podra cambiar su contenido interno

Dejan de ser funciones nativas de JS para empezar a comportarse como clases

        class Manolo extends Component{         // aca usamos las palabras claves class y extends y usamos el 
                                                    Component que lo requerimos en la primer linea junto a React

            render(){                           // utilizamos el metodo render(){}

                return(                         // y adentro del render usamos el return () con las etiquetas de
                                                lo que vayamos a usar

                )
            }
        }
